# File Organizer by Extension

## Description
A Bash script that organizes files in the home directory based on their extensions.

The script processes files only and ignores directories.

## How It Works
- Scans files in the home directory
- Creates a main folder:
  ~/folder-extension
- Creates one folder per extension
- Moves files to their corresponding folders
- Files without extension are moved to:
  no-extension

## Features
- Ignores directories
- Handles files without extensions
- Automatic folder creation
