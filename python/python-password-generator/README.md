# Password Generator

## Description
A simple password generator that creates passwords based on user-selected difficulty.

The program focuses on clarity and basic logic rather than advanced security features.

## Difficulty Levels
- Easy: Letters only
- Medium: Letters and numbers
- Hard: Letters, numbers, and symbols

## User Input
- Password length
- Difficulty level

## Output
- A randomly generated password

## Skills Demonstrated
- Random selection
- Loops
- Conditional statements
- User input handling
